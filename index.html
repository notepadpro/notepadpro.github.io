Welcome

Notepad Pro is a generic text editor that allows you to create,   open, and read plaintext files. If the file contains special formatting or is not a plaintext file, you will not be able to be read in Notepad Pro. The image to the right is a small example of what the Notepad Pro may look like while running.

Note: If you need built-in features such as multimedia  (e.g., pictures), etc. you should be using a word processor and  not an editor.

Supporting OS

All Windows operating system
