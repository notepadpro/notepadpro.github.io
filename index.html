Welcome


Notepad Pro is a generic text editor that allows you to create, open, and read plaintext files.

If the file contains special formatting or is not a plaintext file, you will not be able to be read in Notepad Pro.



Special features

1.Supports both 32-bit and 64-bit OS

2. Enable to detect URLs

3. Print Preview - Print Preview allows you to see the page layout and file contents before printing.

4. Font case - You can change the selected font into the upper or lower case from the Font menu

5. Date/Time - You can insert date and time together or separately from the Insert menu.

6. Zoom - You can easily Zoom In or Zoom Out the page contents from the View menu



Note: If you need built-in features such as multimedia  (e.g., pictures), etc. you should be using a word processor and  not an editor.
