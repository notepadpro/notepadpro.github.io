Welcome

Notepad Pro is a generic text editor that allows you to create, open, and read plaintext files.


Special features

1. Supports both 32-bit and 64-bit OS.

2. Enable to detect URLs

3. Zoom - You can easily Zoom In or Zoom Out the page contents from the View menu

4. Date/Time - You can insert date and time together or separately from the Insert menu.

5. Symbols - Let you insert different kind of Symbols in the document.


Supporting OS

Windows Vista/Server 2008/7/8/8.1/10


Additional Requirments

Microsoft .NET Framework 4.5 Download


Note : If you need built-in features such as multimedia (e.g., pictures), etc. you should be using a word processor and not an editor.
