Welcome

Notepad Pro is a generic text editor that allows you to create, open, and read plain text files.


Special features

1. Enable to detect URLs.

2. Spell Check - Spell Check function is available for checking spelling mistakes.

3. Zoom - You can easily Zoom In or Zoom Out the page contents from the View menu.

4. Date/Time - You can insert the date and time together or separately from the Insert menu.

5. Symbols - Let you insert different kinds of Symbols in the document.


Supporting OS

Windows Vista / 7 / 8 / 8.1 / 10 / 11


Additional Requirments

Microsoft .NET Framework 4.5 Download

Note : If you need built-in features such as multimedia (e.g., pictures), etc, you should be using a word processor and not a text editor.
